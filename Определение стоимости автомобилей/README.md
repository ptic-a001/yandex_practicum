# Описание 
Сервис по продаже подержанных автомобилей «Не бит, не крашен» разрабатывает приложение для оценки рыночной стоимости автомобилей, используя исторические данные о технических характеристиках, комплектациях и ценах. Цель проекта — создать модель машинного обучения, которая быстро и точно определяет стоимость автомобиля. Основные требования заказчика к модели включают высокое качество предсказания, минимальное время обучения и скорость выполнения предсказаний, при этом значение RMSE должно быть меньше 2500. В процессе разработки предлагается исследовать различные типы моделей, включая, но не ограничиваясь, градиентным бустингом с использованием библиотеки LightGBM.
# Вывод 
В заключение проекта по созданию модели определения стоимости автомобилей для сервиса «Не бит, не крашен», можно подчеркнуть следующие ключевые моменты

## Проделанная работа по предобработке: 
Была проведена обширная работа по очистке данных, включая удаление аномальных значений по мощности и дате регистрации, корректировку ошибок в годах регистрации, и замену отсутствующих значений в типе топлива.
- был проведен анализ появления пропущенных значений в следствии чего при возможности было восстановлены значения, также анализ на возможность использования алгоритмов восстановления данных на этом датасете показал, что данные могут получить много шума при таком подходе, в следствии чего остальная часть пропусков была удалена
- был проведен анализ на адекватность данных, вследствие чего были выявлены аномальные значения
- для уменьшения зашумленности были удалены из рассмотрения данные которые противоречили оффициальным данным собранным из интернета 
## Итоги работы модели
- Была выполнено обучение модели на сырых данных для оценки корректности проделанной предобработки и EDA
    - Модель бустинга: RMSE: 1946.47 
    - Модель линейной регрессии: RMSE: 3177.71 
-  Метрика RMSE после проведенной работы с данными:
    - Модель бустинга:
        - RMSE: 1625.61
        - Время обучения: 5.12c
    - Модель линейной регрессии:
        - RMSE: 2398.20
        - Время обучения: 1.23c
- Если важна скорость, то линейная регрессия укладывается в допустимую погрешность и выполняется почти в 5 раз быстрее, но если важна точность то бустингу удалось почти на 1000 сузить погрешность.
- RMSE меньше порога почти на 700евро.
